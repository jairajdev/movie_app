## Installation

npm install or yarn

## Android
react-native run-android or yarn android

## IOS
npx pod-install

react-native run-ios or yarn ios

## Before Run
- Create an api key in [themoviedb](http://themoviedb.org)
- Create .env file and add this line ( API_KEY=YOUR_THEMOVIEDB_API_KEY )
